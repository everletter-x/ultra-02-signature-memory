# EverLetter Ultra 02 - Signature Memory

The flagship template — fully custom layout based on customer requests. Most flexible and premium of all EverLetter templates.

## Tech Stack

- Next.js 13
- TypeScript
- Tailwind CSS
- Framer Motion

## Features

- Gold accent theme on deep black background
- Dynamic section rendering from `config.json`
- Premium animations: parallax, fade, scale, blur transitions
- Music floating button
- Fully data-driven — zero hardcoded content

## Sections

| Type | Description |
|------|-------------|
| `hero` | Full-screen with large title, subtitle, elegant animation |
| `story` | Narrative text with beautiful typography |
| `photos` | Photo grid/carousel with captions |
| `message` | Personal message with special styling |
| `closing` | Final emotional moment |

## Getting Started

```bash
pnpm install
pnpm dev
```

## Configuration

Edit `public/config.json` to customize all content, sections, photos, music, and captions.
